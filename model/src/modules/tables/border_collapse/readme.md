Utilities for controlling whether table borders should collapse or be separated.

## Patterns

| Class             | Css                         | 
|-------------------|-----------------------------| 
| `border-collapse` | `border-collapse: collapse` |
| `border-separate` | `border-collapse: separate` |

## Arbitrary

- `border-collapse-[X]`

```shell
border-collapse: [X]
```
