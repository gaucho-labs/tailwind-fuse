Utilities for controlling how a background image behaves when scrolling.

## Patterns

| Class       | Properties                       |
|:------------|:---------------------------------|
| `bg-fixed`  | `background-attachment: fixed;`  |
| `bg-local`  | `background-attachment: local;`  |
| `bg-scroll` | `background-attachment: scroll;` |
